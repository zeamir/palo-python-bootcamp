1-Day Schedule
========================

- 09:30 - 09:45: Introduction (15m)
    - 5m Introduction about me
    - 5m Participant impressions and expectations
    - 5m AI at work and its effect on programming needs

- 09:45 - 10:00: Repo & project setup (15m)
    - Clone https://github.cyberng.com/azahavi/python-bootcamp or if no access
    - https://github.com/zeamir/palo-python-bootcamp
    - Setting up poetry project

- 10:00 - 10:45: Core Python Foundations (45m)
    - List, generator, and dict comprehensions
    - Function arguments (forcing named args, **kwargs)
    - Mutable default parameters (Common pitfalls)

- 10:45 - 10:55: Break (10m)

- 10:55 - 11:15: Practical Lab (20m)

- 11:15 - 12:00: Modern Python & Typing (45m)
    - Type hints
    - Pydantic (Also install PyCharm plugin for it)

- 12:00 - 12:15: Practical exercise - Pydantic (15m)

- 12:15 - 12:25: Break (10m)

- 12:25 - 13:10: Advanced Python Concepts (45m)
    - Context managers (logger context)
    - Decorators (init logger, Retry)
    - Dunder methods (and why are they called Dunder methods)
    - Cache (lru_cache vs. Cachetools)

- 13:10 - 14:10: Lunch (1h)

- 14:10 - 15:10: Testing & Performance (1h)
    - Layers for testing code
    - Unit tests (python mockito)
    - Fixtures

- 15:10 - 15:25: Break (15m)

- 15:25 - 16:10: Practical session on unit tests (45m)

- 16:10 - 16:25: Summary and open discussion (15m)
